Comencé a alimentar una paloma. Un minuto más tarde, llegaron dos volando, y un minuto más tarde llegaron otras tres. Luego cuatro, y así sucesivamente (Ej: 1 2 3 4 + ...). Una porción de comida alimenta una paloma por un minuto. Si la comida no es suficiente para todas las aves, las palomas que llegaron más temprano comen primero. Las palomas son animales hambrientos que comen y no saben cuando parar. Si tengo N porciones de comida, ¿cuántas palomas se alimentan con al menos una porción de trigo?

Entrada: Número (entero positivo) de porciones de trigo.

Saída: Número (entero positivo) de palomas alimentadas.

Ejemplos: 

checkio(1) == 1
checkio(2) == 1
checkio(5) == 3
checkio(10) == 6

Pistas: Esta tarea muestra cómo podemos usar la programación para modelar diversas situaciones. Está claro que el modelo tiene limitaciones, pero a menudo no es necesario un modelo perfecto.

Precondiciones: 0 < N < 105.
